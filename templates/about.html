About

The SHARED project is an international collaboration building a database of the viral sequences, treatment histories, and health outcomes related to the hepatitis C virus.

SHARED is based in Vancouver, Canada, at the University of British Columbia.

For more details about the SHARED project's candidate cohort, objectives, and participants, see the SHARED Study Protocol. For more details about the information in the database, consult the SHARED Database Manual.
