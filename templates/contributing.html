Submit Data to SHARED

This page outlines the process for contributing data to SHARED. You can find more technical details in the SHARED Database Manual. For more details on the data sharing agreement, contact the project coordinator.

Data Sharing Process

  1. Before starting the data submission process, you'll need ethics approval to share your data and a data sharing agreement with UBC (where where the project is hosted).
  2. To facilitate data merging and minimise ambiguity, we ask that you put your data in a standardised format.
    • For simpler studies, where there's only a single record per participant, we ask that your data conforms to the flat submission scheme .
    • For more complex studies, we ask you to use the multi-table submission scheme .
  3. Once your data is formatted, create a .ZIP file with your patient data and viral sequences.
  4. The SHARED project coordinator will send you a unique data submission link. Follow this link to a web-page where you can securely submit your data.
